Shri Krishna Udhav Memorial Trust Website
========================================

A responsive and modern website for Shri Krishna Udhav Memorial Trust, a registered NGO in Deoria, Uttar Pradesh, focused on social welfare, education, health, and inclusive community development.

1. ABOUT THE PROJECT
---------------------------------------------------------
This website provides detailed information about the trust, its mission, vision, objectives, events, gallery, and donation process. 
It also allows users to contact the trust directly and access important links and policies.
The design is fully responsive and works on both desktop and mobile devices.

---------------------------------------------------------
2. FEATURES
---------------------------------------------------------
- Fully responsive design for desktop and mobile
- About, Mission & Vision, and Objectives sections
- Recent Events and Gallery Highlights with images
- Team / Trustees section
- Donate Now form with multiple payment options and tax benefit details
- Quick Contact Information with social media links
- Footer with Important Links, DARPAN & registration info
- Back to Top button for easy navigation
- SEO-friendly with meta tags and schema.org structured data

5. INSTALLATION & LOCAL SETUP
---------------------------------------------------------
1. Download or clone this repository to your local machine:
   git clone https://github.com/YourUsername/ShreeKrishnaUdhavTrust.git

2. Navigate to the project folder:
   cd ShreeKrishnaUdhavTrust

3. Open with Live Server (recommended) or any local server:
   - VS Code: Right-click index.html → Open with Live Server
   - Python HTTP server:
       python -m http.server 8000
       Then open http://localhost:8000

4. Opening directly in browser (file:///) works, but some fonts or icons may behave differently on mobile viewport.
