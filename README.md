Site Folder Structure :
------------------------
ngo_website/
├── index.html
├── styles.css
├── script.js
├── README.txt
└── images/
    ├── logo.jpg (your circular logo)
    ├── tricolor-banner.jpg (your Vande Mataram/tricolor)
    ├── founder.jpg (founder photo)
    ├── signboard.jpg (location signboard photo)
    ├── event-banner.jpg (event or yellow banner)
    └── team-photo.jpg (volunteer or group photo)

How to Launch:
---------------
1. Place all files above as shown. 
2. Add your actual images to the images/ folder with matching names.
3. Edit text in HTML as needed (e.g., insert your bank/UPI details and accurate office bearer names).
4. Upload to Netlify, Vercel, or any standard web hosting.
5. Your NGO-compliant, bilingual, professional trust website is LIVE!
